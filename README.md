#### Синтаксис
./bin/hexlet-path-size [OPTION]... [FILE]...
 
#### Опции         
-a, --all: Включает скрытые файлы и папки в расчёт

-H, --human: Выводит размеры в человекочитаемом формате (Например: 1K 234M 2G)

-r, --recursive: Позволяет рекурсивно обходить вложенные директории при рассчёте

-h, --help: Выводит текст подсказки
